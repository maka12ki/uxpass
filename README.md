# UXPASS v0.9.1

UX検定（基礎）学習アプリの差分改善版です。

## 今回の主な改善

- ホームのロードマップ表示を改善
- 合格予測のロジックを修正
- 章末チェック方式へ変更
- 問題演習に章・ジャンル選択を追加
- ミニ模試の採点バグを修正
- 模試専用ケース問題を追加

## GitHub / Netlify 更新方法

1. ZIPを解凍
2. GitHub の `uxpass` リポジトリで `Add file` → `Upload files`
3. 解凍した中身を上書きアップロード
4. `Commit changes`
5. Netlify が自動デプロイします
