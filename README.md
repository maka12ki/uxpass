# UXPASS v0.8 GitHub Ready

UX検定（基礎）対策アプリのGitHub/Netlify公開用スターターです。

## 使い方
1. このフォルダの中身をGitHubリポジトリ `uxpass` にアップロード
2. NetlifyでGitHub連携
3. Publish directory は `/` のまま

## v0.8の方針
- 数より問題品質を優先
- ケース問題・比較問題を中心に設計
- 進捗保存は localStorage
- データは `js/app.js` 内に収録（Netlifyで安定動作）
